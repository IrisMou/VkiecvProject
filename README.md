# 威客竞技-专业竞技预约联盟

  应用介绍: "威客竞技-专业竞技预约联盟"app是一款预约网咖和购买电竞商品的应用。

  核心功能: 预约+商城。
  
    商城：电竞耳机/鼠标、电竞游戏手柄、电子散热器等；
         核心流程：线上下单 - 凭订单编号 - 到店取货。
          
    场馆：电竞网咖馆；
          预定后需在预约日期到达对应场馆进行消费，逾期订单将自动取消。
          
    首页：精选商品、场馆优选、今日特价限时抢购、本周单品推荐、推荐场馆和猜你喜欢等。
    
    订单：商品订单、我的预约等。
    
    我的：商品收藏、场馆收藏以及相关设置等。
